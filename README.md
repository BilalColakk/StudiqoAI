<a name="turkish"></a>
## 🇹🇷 Türkçe

Studiqo, öğrencilerin sınav hazırlık süreçlerini yapay zeka ile optimize eden modern, oyunlaştırılmış bir çalışma planlama platformudur.

### ✨ Özellikler
- **🧠 AI Planlama:** Derslerinize ve müsaitlik durumunuza göre size özel haftalık çalışma planları oluşturur.
- **🔥 Seri (Streak) Sistemi:** Her gün çalışarak serinizi bozmadan ilerleyin ve motivasyonunuzu koruyun.
- **🏅 Başarımlar (Badges):** Hedeflerinize ulaştıkça rozetler kazanın ve profilinizde sergileyin.
- **📊 Verimlilik Analizi:** Odak skorunuzu ve çalışma istatistiklerinizi detaylı grafiklerle takip edin.
- **🎯 Odak Modu:** Çalışırken dikkatinizi dağıtacak her şeyi gizleyen minimalist odak ekranı.
- **🌓 Tema Desteği:** Dark ve Light mode seçenekleriyle konforlu kullanım.
- **📱 Mobil Uyumlu:** Telefonunuzdan veya tabletinizden planlarınıza her an erişin.

### 🛠️ Kurulum

#### Backend
1. `backend` klasörüne gidin.
2. `.env.example` dosyasını `.env` olarak kopyalayın ve ayarları yapın.
3. Bağımlılıkları yükleyin: `pip install -r requirements.txt`
4. Sunucuyu başlatın: `uvicorn app.main:app --reload`

#### Frontend
1. `frontend` klasörüne gidin.
2. `.env.example` dosyasını `.env` olarak kopyalayın.
3. Bağımlılıkları yükleyin: `npm install`
4. Uygulamayı başlatın: `npm run dev`

<a name="english"></a>
## 🇺🇸 English

Studiqo is a modern, gamified study planning platform that optimizes students' exam preparation processes with artificial intelligence.

### ✨ Features
- **🧠 AI Planning:** Creates customized weekly study plans based on your courses and availability.
- **🔥 Streak System:** Stay motivated by maintaining your daily study streak.
- **🏅 Achievements (Badges):** Earn badges as you reach your goals and display them on your profile.
- **📊 Productivity Analysis:** Track your focus score and study statistics with detailed charts.
- **🎯 Focus Mode:** A minimalist screen that hides distractions while you study.
- **🌓 Theme Support:** Comfortable use with Dark and Light mode options.
- **📱 Mobile Friendly:** Access your plans anytime from your phone or tablet.

### 🛠️ Setup

#### Backend
1. Go to the `backend` folder.
2. Copy `.env.example` to `.env` and configure settings.
3. Install dependencies: `pip install -r requirements.txt`
4. Start server: `uvicorn app.main:app --reload`

#### Frontend
1. Go to the `frontend` folder.
2. Copy `.env.example` to `.env`.
3. Install dependencies: `npm install`
4. Start app: `npm run dev`
